## Description

[Nest](https://github.com/nestjs/nest) project for a simple API to manage CRUD of users and utilize BullMQ to send emails (console logs in this case).

## Project setup

```bash
$ npm install
```

## Run Build
```bash
$ npm run build
```

## Run migrations

```bash
$ npm run migration:run
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
